# 采购审批矩阵

## 标准路径
满足以下条件时，项目可以按标准路径推进：

1. 供应商基础材料齐备。
2. 未涉及敏感数据处理。
3. 合同未触碰核心红线。
4. 无需额外例外审批。

## 必须升级法务的情形
出现下列任一情形时，采购不得直接推进，必须升级法务审查：

1. 删除或弱化责任上限。
2. 删除审计权、便利终止权或保密义务。
3. 修改赔偿责任、争议解决或适用法律条款。
4. 对方回传了非我方标准合同版本或大幅修改版。

## 必须升级安全复核的情形
出现下列任一情形时，采购应补充安全复核：

1. 供应商将接触客户数据、员工数据或业务数据。
2. 供应商需要接入公司系统、接口或生产环境。
3. 供应商的数据存储地点不清楚。
4. 供应商存在分包安排但未说明边界。
