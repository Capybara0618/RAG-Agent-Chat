### 前后端互连
【启动阶段】
1. 启动 FastAPI
2. 创建 app
3. 执行 lifespan
4. 创建 container
5. app.state.container = container
6. 后端开始监听端口

7. 启动 Streamlit
8. 前端页面渲染出来

------------------------------------------------

【用户点击“开始提问”】

9. 用户在前端输入 query
10. 用户点击按钮
11. 前端调用 api_post("/chat/query", json=...)
12. 前端发出 HTTP POST 请求

------------------------------------------------

【后端接收请求】

13. Uvicorn 接收请求
14. FastAPI 为本次请求创建 Request 对象
15. FastAPI 找到匹配路由 query_chat

------------------------------------------------

【参数准备】

16. 解析 JSON，构造 payload
17. 执行 get_container(request)
18. 从 request.app.state.container 取出 container
19. 执行 get_agent_service(container)
20. 拿到 agent_service
21. 执行 get_db(container)
22. 创建 db session
23. 把 db yield 给路由函数

------------------------------------------------

【业务执行】

24. 执行 query_chat(payload, db, agent_service)
25. 进入 agent_service.query(db, payload)
26. 做检索 / 调用模型 / 查数据库 / 组装结果
27. 返回 QueryResponse

------------------------------------------------

【请求结束】

28. FastAPI 把结果转成 JSON 响应
29. yield 依赖收尾，db.close()
30. 本次 Request 生命周期结束

------------------------------------------------

【前端显示结果】

31. 前端拿到 response.json()
32. 保存 session_id / trace_id 到 st.session_state
33. 把 answer/citations 渲染到页面
34. 用户看到结果
