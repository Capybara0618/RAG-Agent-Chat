# 供应商回传修订版协议

## 核心条款
责任上限调整为不超过三个月服务费总额。
乙方仅退还已收费用，不承担其他赔偿责任。
乙方可根据运营需要将服务数据传输至其关联部署地点，并可自行安排分包。
如发生安全事件，乙方将在合理可行范围内尽快通知，但不承诺固定通知时限。
乙方可向合作方披露业务信息用于服务运营。
